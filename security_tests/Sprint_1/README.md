# Evidencias Security Tests — Sprint 1

Carpeta destinada a registrar escaneos y validaciones ofensivas controladas para readiness.

## Contenido esperado

- Enumeración de puertos por zona.
- Validación de exposición esperada/no esperada.
- Evidencia de hallazgos o confirmación de mitigación.

## Convención de nombre

`YYYYMMDD-HHMM_<zona>_<tipo-prueba>_<responsable>.<ext>`

Ejemplo: `20260329-2040_dmz_nmap_lucas.txt`
