# Evidencias QA — Sprint 1

Carpeta destinada a almacenar evidencia de validación funcional de readiness.

## Estructura sugerida

- `logs/` (salidas de comandos)
- `capturas/` (screenshots)
- `resultados/` (resúmenes de pruebas)

## Convención de nombre

`YYYYMMDD-HHMM_<ID-EVIDENCIA>_<zona>_<responsable>.<ext>`

Ejemplo: `20260329-2015_EV-S1-003_dmz_catalina.txt`

## Relación obligatoria

Cada archivo debe estar referenciado en:

- `docs/SPRINT_1/Evidencia_Pruebas_Basicas.md`
- `docs/SPRINT_1/Informe_Readiness.md`
