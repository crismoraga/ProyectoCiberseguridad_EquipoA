# Evidencia Técnica — Pruebas Básicas Sprint 1

> Objetivo: asegurar trazabilidad entre requisito, prueba ejecutada, resultado y artefacto de evidencia.

## Formato mínimo de evidencia
- **ID evidencia**
- **Fecha/hora**
- **Responsable**
- **Requisito asociado**
- **Comando o acción ejecutada**
- **Resultado esperado**
- **Resultado obtenido**
- **Ruta del archivo (captura/log/salida)**
