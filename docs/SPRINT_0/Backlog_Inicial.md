## Definición de “Hecho” por tarea técnica
Una tarea técnica se considera cerrada cuando tiene:
- Cambio implementado en infraestructura/configuración.
- Validación técnica mínima ejecutada.
- Evidencia almacenada con fecha/responsable.
- Actualización documental del artefacto correspondiente.
